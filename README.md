# sobre o projeto

API para gerenciamento bancário da instituição Okay

# Features

- Logar pelo Cpf e Senha
- Procurar pela agência e conta
- Transferir saldo

# Tecnologias

- Java 17
- JPA hibernate
- Maven
- MySQL 
- Spring boot
- Junit5
- Spring Dev Tools
- Spring Security 
- Swagger

# Execução do Projeto
Pré-requisito: Java 17

```bash
# clonar repositório
git clone https://github.com/abnerjosefelixbarbosa/api-banco-okay.git
```
