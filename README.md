# About project

Back-end API which allows to do transactions, where the customer enters his profile and selects the balance transfer operation to another account.

# Features

- Logar pelo Cpf e Senha
- Procurar pela agência e conta
- Transferir saldo

## Back-end

- Java 17
- JPA hibernate
- Maven
- MySQL 
- Spring boot
- Junit5
- Spring Dev Tools
- Spring Security 
- Swagger
- Lombok

# Project Execution
